Parágrafos

O elemento HTML <p> (ou elemento de parágrafo) representa um parágrafo do texto.


Listas

O elemento HTML <ul> (ou elemento de lista desordenada) representa uma lista de itens sem ordem rígidas, isto é, uma coleção de itens que não trazem uma ordenação númerica e as sua posições.

  • Banana
    • Nanica
    • Prata
  • Abacaxi
  • Graviola

O elemento <ol> ( ou elemento de lista ordenada) representa uma lista de itens ordenados.

  1. Banana
  2. Abacaxi
  3. Graviola

O elemento HTML <dl> (ou elemento de lista de definições) engloba uma lista de pares de termos e descrições.

Textos pré-formatados

O elemento HTML <pre> representa o texto pré-formatado que deve ser apresentado exatamente como está escrito no arquivo HTML.

        *   TEXTO PRÉ-FORMATADO    *
