Leiaute


Contêiners

Os contêiners são os elementos de leiaute mais básicos no Bootstrap e são usados para conter, preencher e (às vezes) centralizar o conteúdo dentro deles.

Por padrão o Bootstrap estabelece os pontos de quebra(media querry) para alguns tamanhos de telas/janela.

Exemplos:

.container

.container-sm

.container-md

.container-lg

.container-xl

.container-fluid
